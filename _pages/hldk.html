The Higher Level Dev Kit

To install the dev kit, please visit: https://github.com/HigherLevelDev/hldk

You will need an API Key to run it, which you can get here.

Summary of the Dev Kit

The Higher Level Dev Kit (HLDK) is a powerful tool that allows you to develop at a higher level using natural language. It offloads the actual coding to an LLM-powered Software Engineering Agent.

Key Features:

  • Node-based server and web UI
  • Supports Anthropic Claude Sonnet 3.5 (and also OpenAI & Ollama)
  • Audio transcription support (talk instead of type)
  • Workspace management for multiple projects
  • Git integration for version control
  • Automatically runs and fixes tests

Getting Started:

  1. Set up your Anthropic API key
  2. Install required tools (Node.js 21+ and pnpm)
  3. Set up audio transcription (optional)
  4. Run the HLDK server
  5. Access the web UI and import your local git repo as a workspace

For detailed instructions and more information, please refer to the GitHub repository.
